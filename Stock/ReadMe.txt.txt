Реализовать клиент-серверную систему распространения котировок подписчикам.

Trading инструменты:
RUB_EUR
EUR_USD
USD_CHG
USD_AUD
USD_GBP

Компонент "Источник котировок":
 - генератор случайных чисел, раз в 0.6 секунды выдает новое значение одной из 5 котировок.

Компонент "Сервер":
 - хранит последние значения котировок, поставляемых источником котировок
 - осуществляет прослушивание socket, ждет подключения клиентов к TCP port 3333
 - при подключении нового клиента передает ему все имеющиеся текущие котировки
 - раз в 2 секунды отправляет всем подключенным клиентам только изменившиеся котировки
 - принимает и обрабатывает команды клиента (действуют только на того клиента, который командовал):
   q - закрывает соединение
   r - "row style" (default) - передает котировки строкой: RUB_EUR=1.23;USD_CHG=0.78
   c - "column style" - передает котировки столбцом:
RUB_EUR=1.23
USD_CHG=0.78

Компонент "клиент":
 - подключается к TCP port 3333 сервера.
 - выводит на экран получаемые котировки
 - предоставляет возможность отправлять серверу команды (q,r,c)

В зависимости от предположительного общего estimate всего проекта (не более 8 часов)
можно не делать реализацию клиента, а просто использовать стандартный telnet клиент
(в Windows 7 по умолчанию отключен, добавляется через "Включение или отключение компонентов Windows").

Необходимо реализовать возможность тестового запуска системы с 4 клиентами и 1 сервером.


Оцениваться будет общий стиль, наличие необходимых тестов, применение паттернов проектирования.
Так же просьба указать фактическое количество затраченного времени.
